# klingeLING!
Time tracker. Full featured. Including Harvest import.

## This is beta software.

klingeLING! is my first vibe coded project. PRD was done with Claude chat while driving the Autobahn. Then the PRD was pasted into the Claude chat in VS Code with autonomous agent enabled. The code was generated by the agent and then refined via chat. Not a single line of code was written by me. 

Except for this README.

This is mind blowing.

## Features

- Manual time entry with project and task selection
- Clients, project and tasks
- Hourly rates based on users, clients, projects and tasks
- Time tracking reports
- Harvest import
